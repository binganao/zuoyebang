## 作业帮抢题助手

由于时间问题，无法继续开发这一脚本，故将其开源。

### 结构说明

```
source：源码
ico：图标
backend：后端
```

### 运行方式

**脚本：**

```bash
pip install -r requirements.txt
python zyb.py
```

**后端：**

将后端代码放入PHP服务端内。

**构建：**

```bash
pip install pyinstaller
pyinstaller -F zyb.py --i zuoye.ico
```
